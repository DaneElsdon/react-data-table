# React Data Table Challenge

A reusable React data table component built for the frontend interview challenge.

## Features

- Displays user data in a responsive table
- Search/filter by ID, name, email, or role
- Sort users by ID, name, email, or role
- Empty state when no users are found
- Highlights matching search text in the table
- Responsive Bootstrap layout
- Role badges with custom styling

## Tech Stack

- React
- Vite
- Bootstrap
- CSS

## Setup Instructions

### 1. Clone the repository

git clone YOUR_REPOSITORY_URL

### 2. Navigate into the project directory

cd react-data-table

### 3. Install all project dependencies

npm install

### 4. Start the development server

npm run dev

### 5. Open the application in your browser

Open the local development URL shown in the terminal (usually http://localhost:5173).

### 6. Build the project for production

npm run build

### 7. Preview the production build locally

npm run preview
